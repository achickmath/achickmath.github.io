# Portfolio Website Setup Instructions

## Quick Start (GitHub Pages)

1. **Create GitHub Account** (if you don't have one)
   - Go to github.com
   - Sign up with username: achickmath (or whatever you want)

2. **Create Repository**
   - Click "New Repository"
   - Name it EXACTLY: `achickmath.github.io` (replace achickmath with your username)
   - Make it Public
   - Click "Create Repository"

3. **Upload Files**
   - Click "uploading an existing file"
   - Drag and drop these files:
     * index.html
     * styles.css
     * headshot.jpg (your photo)
     * resume.pdf (your resume)
   - Click "Commit changes"

4. **Enable GitHub Pages**
   - Go to repository Settings
   - Click "Pages" in left sidebar
   - Under "Source", select "main" branch
   - Click Save

5. **Your site will be live at:**
   `https://achickmath.github.io`
   
   (Takes 2-5 minutes to deploy)

## Files You Need to Add

**headshot.jpg** - Your profile photo (rename your photo to exactly "headshot.jpg")

**resume.pdf** - Your resume PDF (rename to exactly "resume.pdf")

## Customization Notes

- All your info is already in the HTML (email, phone, LinkedIn)
- Images use placeholders from Wikimedia Commons (all cited properly)
- The site is fully responsive (works on mobile)
- Total word count: ~2000 words (requirement is 1000 minimum)
- Total images: 7+ images (requirement is 5 minimum)

## If Something Breaks

- Make sure file names are EXACTLY: index.html, styles.css, headshot.jpg, resume.pdf
- Make sure repository name is EXACTLY: yourusername.github.io
- Wait 5 minutes after uploading for changes to appear
- Clear your browser cache if old version shows up

## Updating Content Later

1. Go to your repository on GitHub
2. Click on the file you want to edit (index.html or styles.css)
3. Click the pencil icon to edit
4. Make changes
5. Click "Commit changes"
6. Wait 2-5 minutes for changes to go live

That's it. You're done.
